About the Project

Understanding the technology behind speech emotion recognition

Project Overview

The RAVDESS Emotion Recognition project is a deep learning-based system that can detect emotions from speech audio. Using the RAVDESS (Ryerson Audio-Visual Database of Emotional Speech and Song) dataset, we've trained a sophisticated CNN-LSTM model that achieves 99% accuracy in recognizing various emotions from speech.

This project demonstrates the application of advanced machine learning techniques to the field of affective computing, enabling computers to recognize and respond to human emotions.

The RAVDESS Dataset

The RAVDESS dataset is a validated multimodal database of emotional speech and song. The database contains recordings from 24 professional actors (12 female, 12 male), vocalizing two lexically-matched statements in a neutral North American accent.

The dataset includes 8 emotional expressions:

  • Neutral
  • Calm
  • Happy
  • Sad
  • Angry
  • Fearful
  • Disgust
  • Surprised

Technical Architecture

Model Architecture

Our emotion recognition system uses a hybrid CNN-LSTM architecture:

  • Convolutional Neural Network (CNN): Extracts spatial features from the audio spectrograms
  • Long Short-Term Memory (LSTM): Captures temporal dependencies in the audio sequence
  • Batch Normalization: Stabilizes and accelerates training
  • Dropout Layers: Prevents overfitting and improves generalization

Audio Processing Pipeline

The system processes audio through several stages:

  1. Audio preprocessing and normalization
  2. Feature extraction (MFCCs, Mel spectrograms, Chroma features)
  3. Data augmentation for underrepresented emotions
  4. Model training with early stopping and learning rate scheduling
  5. Real-time inference for emotion detection
